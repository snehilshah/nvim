# Neovim

### Motions {.break-inside-avoid}

| Shortcut    | Description              |
| ----------- | :----------------------- |
| `<Space-x>` | close the current window |
| `<C-n>`     | open file tree           |
| `<C-h>`     | focus file tree          |
| `<C-n>`     | open file tree           |

## commands

| Shortcut          | Description                         |
| ----------------- | :---------------------------------- |
| `:Lazy`           | Opens the lazy package manager      |
| `:echo &filetype` | print the filetype of current file  |
| `:echo &filetype` | print the filetype of current file  |
| `:set filetype=x` | set the file type to the x language |

## Lazy Plugins

| cmd            | Description               |
| -------------- | :------------------------ |
| `BufEnter`     | before opening the file   |
| `BufWritePost` | after saving the file     |
| `InsertLeave`  | after exiting insert mode |

## Customizations

| shortcut      | Description  |
| ------------- | :----------- |
| `<space-t-h>` | change theme |

## File Tree

| shortcut | Description                          |
| -------- | :----------------------------------- |
| `a`      | create new file/folder               |
| `d`      | delete the current file              |
| `D`      | thrash the current file (recyle bin) |

## Code Editing

| shortcut     | Description                          |
| ------------ | :----------------------------------- |
| `<leader-/>` | comment the current line             |
| `d`          | delete the current file              |
| `D`          | thrash the current file (recyle bin) |
| `<C-e>`      | go to the last line in normal mode   |

## Git

| symbol | meaning                              |
| ------ | :----------------------------------- |
| `⭐`   | new file                             |
| `❌`   | changed file                         |
| `D`    | thrash the current file (recyle bin) |
