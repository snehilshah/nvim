# Neovim

### Motions {.break-inside-avoid}

| Shortcut    | Description              |
| ----------- | :----------------------- |
| `<Space-x>` | close the current window |
| `<C-n>`     | open file tree           |

## commands

| Shortcut | Description                    |
| -------- | :----------------------------- |
| `:Lazy`  | Opens the lazy package manager |
| `<C-n>`  | open file tree                 |

## Lazy Plugins

| cmd            | Description               |
| -------------- | :------------------------ |
| `BufEnter`     | before opening the file   |
| `BufWritePost` | after saving the file     |
| `InsertLeave`  | after exiting insert mode |
